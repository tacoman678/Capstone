Introduction:

The goal of my final project was to familiarize myself with aspects of software engineering through creating my own api and web design that I have never seen before, as well as creating something that can strengthen my resume and create a baseline for future projects beyond this one. So, to satisfy these standards I have set for myself, I decided to create a web application that collects a specified number (user input) of a desired product (user input) on amazon via web scraping and displays the product name, price, image, and a number (user input) of relevant reviews pertaining to each listing. To accomplish this project, I have broken down the process into seven steps which are listed in the procedure below:

Procedure:

  1. Choose a programming language to build the api and web framework that is compatible with that api for building the web application.
  2. Use a web scraping library from the language I chose to create an API that scrapes Amazon.com for the desired product and product aspects (title, price, image, and reviews) based on user input.
  3. Use the API created in step 2 to extract the product name, price, image, and relevant reviews from the scraped product listings and create a database that is compatible with your chosen web framework to store those items.
  4. Create and design a webpage that can be used to display/showcase projects like this one. Then create a webpage that can be linked to from the main page that has a form in the web application that allows users to input the desired product and the number of products they want to see.
  5. Connect the form on the web application to the api I created by handling the form submission, validating the input, passing it to my api which will retrieve relevant product data from Amazon.com, process the data so that it is in a displayable format on a webpage, and finally return the results.
  6. Deploy the web application to a web hosting service of my choosing.
  7. Test the web application to ensure it is working as expected.
